**Wire** is A fintech website built with hands-on Figma handman model, UX design.
This project focuses on UI development, component structure, and integration of a Figma design into a working web application.

**Project Goal**

The goal of this project was to reproduce a modern landing page from a Figma design and focus on:

Clean UI implementation
Component-based architecture
Responsive design 
Figma design
Frontend development best practices
